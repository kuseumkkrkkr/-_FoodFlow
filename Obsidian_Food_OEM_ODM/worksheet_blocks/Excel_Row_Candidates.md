## 현재 엑셀에 넣은 1번

| 항목 | 내용 |
|---|---|
| 선택한 식품 트렌드 키워드 | 기능성·개인맞춤 식품 |
| 트렌드의 핵심 의미 | 단백질, 식이섬유, 장건강, 에너지·멘탈 관리처럼 효능이 명확한 제품을 개인 목적에 맞춰 선택·기획하는 흐름 |
| 트렌드가 나타난 배경 | 2026 식품 트렌드에서 단백질, 장건강, 기능성 음료와 개인화 영양이 핵심으로 제시됨. 브랜드는 기능성 원료와 제형에 맞는 빠른 제품화가 필요함 |
| 관련 소비자 변화 | 소비자는 단순한 맛보다 건강 목적, 성분 투명성, 섭취 상황별 편의성을 함께 요구함 |
| 관련 제품·서비스 사례 | 고단백 스낵·음료, 프로/프리바이오틱스 제품, 기능성 RTD 음료, 개인화 영양 앱 |
| 나의 창업 아이디어와의 연결 가능성 | 입력한 제품 컨셉의 기능성 원료, 제형, 인증 조건을 분석해 생산 가능한 OEM/ODM 공장을 추천할 수 있음 |
| 적용 가능한 식품 카테고리 | 음료, 단백질바, 건강간식, 발효식품, 분말/스틱 |
| 아이디어 시사점/메모 | 공장 DB에는 취급 원료, MOQ, 인증, 소량 샘플 가능 여부를 태그화해야 함 |
| 우선 검증 질문 | 기능성 식품 창업자가 공장 탐색에서 가장 오래 걸리는 조건은 무엇인가? |

## 현재 엑셀에 넣은 2번

| 항목 | 내용 |
|---|---|
| 선택한 식품 트렌드 키워드 | AI·투명 공급망 제조 |
| 트렌드의 핵심 의미 | AI와 데이터로 원료, 공정, 인증, 생산 가능 수량, 납기 정보를 연결해 식품 개발·발주를 빠르게 만드는 흐름 |
| 트렌드가 나타난 배경 | AI, 바이오프로세싱, 디지털 추적 기술이 식품 설계·공급망 투명성·안전성에 활용되고, 클린라벨과 책임소싱 요구가 기본 조건으로 확대됨 |
| 관련 소비자 변화 | 소규모 브랜드도 원산지, 제조공정, 클린라벨, 지속가능성을 설명할 수 있는 제조 파트너를 원함 |
| 관련 제품·서비스 사례 | AI 레시피 개발, 스마트 패키징/QR, 추적 가능한 원료 소싱, 소량 생산 D2C 제조 플랫폼 |
| 나의 창업 아이디어와의 연결 가능성 | B2B 대량 발주와 B2C 소규모 제작/공동구매를 분리해 MOQ, 위치, 설비 특성별 최적 공장 매칭이 가능함 |
| 적용 가능한 식품 카테고리 | 소스/드레싱, HMR, 간편식, 베이커리, 펫푸드, 로컬 특산가공 |
| 아이디어 시사점/메모 | 핵심은 공장 역량 데이터 표준화와 견적 자동화. 단순 중개보다 매칭 정확도와 신뢰 검증이 차별점 |
| 우선 검증 질문 | 공장과 발주자가 모두 제공할 수 있는 필수 데이터 항목은 무엇인가? |
