# 혼웰식·한그릇 건강식

## 핵심 의미

1인 가구 증가와 건강지향이 결합해 덮밥, 비빔밥, 샐러드, 샌드위치 같은 원보울·원디쉬 포맷이 성장하는 흐름이다.

## 데이터

- 2024년 1인가구 804.5만 가구, 전체 가구의 36.1%
- 2022~2025년 덮밥 +8.2%, 비빔밥 +13.7%, 샐러드 +22.2% 증가 보도

## 검색/뉴스 신호

검색 직접 순위는 공개되지 않았지만 건강식, 식사, 레시피, 한그릇 메뉴 검색 관심과 연결된다.

## 플랫폼 연결

- 1인분 영양 균형 도시락
- 고단백 샐러드
- 원보울 HMR
- 직장인 식단 구독
- 소량 테스트 생산과 온라인 판매에 적합

## 공장 매칭 조건

| 조건 | 필요 데이터 |
|---|---|
| 제형 | 냉장/냉동 도시락, 샐러드, 덮밥, 소스 동봉 |
| 포장 | 1인분 용기, 소분 소스, 냉장/냉동 배송 가능 |
| 원료 | 단백질 원료, 채소, 곡물, 저당 소스 |
| 인증/표시 | HACCP, 영양성분 산출, 알레르기 표시 |

## 주의점

혼웰식은 트렌드 명칭이므로 실제 제품화 전에는 세부 메뉴, 가격대, 채널 데이터를 별도로 검증해야 한다.
