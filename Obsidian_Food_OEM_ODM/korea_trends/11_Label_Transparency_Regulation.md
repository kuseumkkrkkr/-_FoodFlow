# 클린라벨·영양표시·GMO/카페인 표시

## 핵심 의미

소비자 알권리 강화와 건강/안전 중심 정책 전환으로 라벨링, 원료 이력, 표시 대응이 상품 경쟁력에 포함되고 있다.

## 데이터

- 영양표시 259개 품목 확대
- 과라나 함유 고체식품까지 고카페인 주의문구 확대
- 당알코올 표시 강화
- GMO 완전표시제 등 일부 세부 시행은 하위 규정 확인 필요

## 플랫폼 연결

- 영양성분 자동 산출
- 클린라벨 원료
- 표시 검수 서비스
- 제조 이력 관리

## 공장 매칭 조건

| 조건 | 필요 데이터 |
|---|---|
| 표시 | 영양성분, 당류, 카페인, 당알코올, GMO |
| 증빙 | 원산지, 원료 스펙, 시험성적서, 인증서 |
| 운영 | 표시 변경 대응, 상세페이지 문구 검수 |
| 신뢰 | 소비자 공개 가능 범위, 품질관리 이력 |

## 주의점

GMO 완전표시제 등 일부 세부 시행은 하위 규정 확인이 필요하다.
