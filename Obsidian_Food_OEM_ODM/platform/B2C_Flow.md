# B2C 제작부

## 대상

- 크리에이터
- 로컬 브랜드
- 동아리/커뮤니티
- 소규모 공동구매 운영자
- 테스트 판매를 원하는 예비 창업자

## 주요 흐름

1. 만들고 싶은 식품 입력
2. [[platform/Vibe_Cooking_Digital_Twin|바이브 쿠킹]]으로 생산 가능한 레시피 방향과 원재료 BOM 생성
3. 소량 제작 가능한 카테고리 추천
4. 예상 단가, 최소 수량, 제작 기간 안내
5. 공동구매 또는 사전 예약 수요 확인
6. 샘플 제작 후 소규모 생산

## 차별점

소규모 사용자는 공장과 직접 협상하기 어렵다. 플랫폼이 표준 옵션, 공동구매 수량, 샘플 제작 패키지를 제공하면 진입 장벽이 낮아진다.

바이브 쿠킹은 사용자의 표현을 공장이 이해할 수 있는 개발 사양으로 바꾼다. 예를 들어 "두바이 스타일 저당 쿠키"는 피스타치오 원료, 대체당, 알레르기, 베이커리 공정, 소량 포장, 표시 문구 검토 항목으로 변환된다.
