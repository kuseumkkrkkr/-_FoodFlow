# B2B 제작부

## 대상

- 식품 스타트업
- 유통사 PB 담당자
- 프랜차이즈 메뉴 개발팀
- 브랜드사 상품기획자
- 수출/납품용 제품 개발팀

## 주요 흐름

1. 제품 컨셉 입력
2. [[platform/Vibe_Cooking_Digital_Twin|바이브 쿠킹]]으로 레시피 초안, 원재료 BOM, 공정 조건 생성
3. 카테고리와 공정 조건 자동 분류
4. 인증, MOQ, 납기, 지역 조건 필터링
5. 적합 공장 후보 비교
6. NDA/견적/샘플 발주
7. 본생산 발주

## 필요한 기능

- 공장별 설비와 인증 필터
- 예상 MOQ와 단가 범위
- 견적 요청 템플릿
- 샘플 생산 가능 여부
- 공장 평판/검증 상태
- 원재료 BOM과 대체 원료 비교
- 목표 원가, 목표 소비자가, 마진 조건 시뮬레이션
- 표시·인증·알레르기 리스크 체크리스트
